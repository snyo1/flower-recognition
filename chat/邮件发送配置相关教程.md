### 邮箱配置

1.安装插件

本项目使用邮箱来作为登录和注册的标准，我们使用QQ邮箱来发送验证码。在实际项目中，一般会使用企业邮箱来发送邮件，但是操作方式与个人邮箱类似。另外，针对移动端，正确的做法应该是使用手机号码+短信验证码来进行注册和登录，由于课程篇幅限制，我们就采用邮箱来实现。在FastAPI中发送邮件，我们可以通过`fastapi-mail`插件来实现，安装命令如下：

```sh
$ pip install fastapi-mail
```

FastAPI-Mail的官方文档地址为：https://sabuhish.github.io/fastapi-mail/getting-started/

2.QQ邮箱配置

首先，打开个人QQ邮箱，点击右上角设置，点击左侧账号与安全，点击安全设置，点击“生成授权码”，然后扫码或用绑定的手机号发送短信，发送完成后点击“我已发送”。然后就会生成一个“授权码”，把“授权码”拷贝保存下来。

### 发送邮件

在`core`下创建一个`mail.py`模块，然后填入以下代码，用于创建一个邮箱实例：

```python
# core/mail.py
from fastapi_mail import FastMail, ConnectionConfig
from pydantic import SecretStr, EmailStr
import settings

def create_mail_instance() -> FastMail:
    """创建 FastMail 实例（每次调用返回新实例，线程/协程安全）"""
    mail_config = ConnectionConfig(
        MAIL_USERNAME=settings.MAIL_USERNAME,
        MAIL_PASSWORD=SecretStr(settings.MAIL_PASSWORD),
        MAIL_FROM=settings.MAIL_FROM,
        MAIL_PORT=settings.MAIL_PORT,
        MAIL_SERVER=settings.MAIL_SERVER,
        MAIL_FROM_NAME=settings.MAIL_FROM_NAME,
        MAIL_STARTTLS=settings.MAIL_STARTTLS,
        MAIL_SSL_TLS=settings.MAIL_SSL_TLS,
        USE_CREDENTIALS=True,
        VALIDATE_CERTS=True,
    )
    return FastMail(mail_config)
```

接着将这个创建邮箱实例的代码创建成依赖，方便后期视图函数使用，在`dependencies.py`中添加以下代码：

```python
async def get_mail() -> FastMail:
    return create_mail_instance()
```

测试是否成功

在`main.py`中添加一个用于测试邮件发送的视图函数：

```python
from fastapi import Depends
from dependencies import get_mail
from fastapi_mail import MessageSchema, MessageType, FastMail
from aiosmtplib import SMTPResponseException

@app.get('/mail/test')
async def send_mail_test(
    email: str,
    mail: FastMail = Depends(get_mail)
):
    message = MessageSchema(
        subject="hello",
        recipients=[email],
        body=f"Hello {email}",
        subtype=MessageType.plain
    )
    try:
        await mail.send_message(message)
    except SMTPResponseException as e:
        if e.code == -1 and b"\\x00\\x00\\x00" in str(e).encode():
            print("忽略 QQ 邮箱 SMTP 关闭阶段的非标准响应（邮件已成功发送）")
    return {"message": "邮件发送成功！"}
```

### 发送邮箱验证码

在发送邮箱验证码之前，先创建一个获取操作数据库`session`对象的依赖，在`dependencies.py`中添加以下代码：

```python
from sqlalchemy.ext.asyncio import AsyncSession
from models import AsyncSessionFactory

async def get_session() -> AsyncSession:
    session = AsyncSessionFactory()
    try:
        yield session
    finally:
        await session.close()
```

然后在`repository`下创建一个`user_repo.py`文件，填入以下代码：

```python
from models import AsyncSession
from models.user import User, EmailCode
from sqlalchemy import select, update, delete, exists
from datetime import datetime, timedelta

class EmailCodeRepository:
    def __init__(self, session: AsyncSession):
        self.session = session

    async def create(self, email: str, code: str) -> EmailCode:
        async with self.session.begin():
            email_code = EmailCode(email=email, code=code)
            self.session.add(email_code)
            return email_code

    async def check_email_code(self, email: str, code: str) -> bool:
        async with self.session.begin():
            stmt = select(EmailCode).where(EmailCode.email == email,EmailCode.code == code)
            email_code: EmailCode | None = await self.session.scalar(stmt)
            if email_code is None:
                return False
            if (datetime.now() - email_code.created_time) > timedelta(minutes=3):
                return False
            return True
```

在视图函数中，在某些操作，不需要返回具体数据时，我们可以返回一个固定的响应格式。于是，在`schemas/__init__.py`中填入以下代码：

```python
from pydantic import BaseModel, Field
from typing import Annotated, Literal

class ResponseOut(BaseModel):
    result: Annotated[Literal["success", "failure"], Field("success", description="操作结果")]
```

最后，在`routers`下创建一个`auth_router.py`文件，用于存放和用户操作相关的路由，并添加获取邮箱验证码的视图函数代码：

```Python
import random
import string
from typing import Annotated

from aiosmtplib import SMTPResponseException
from fastapi import APIRouter, Query, Depends, HTTPException
from fastapi_mail import FastMail, MessageSchema, MessageType
from pydantic import EmailStr
from sqlalchemy.ext.asyncio import AsyncSession

from dependencies import get_mail, get_session
from repository.user_repo import EmailCodeRepository
from schemas import ResponseOut

router = APIRouter(prefix="/auth",tags=["user"])


@router.get("/code",response_model=ResponseOut)
async def get_email_code(
        email: Annotated[EmailStr,Query(...)],
        mail: FastMail = Depends(get_mail),
        session: AsyncSession = Depends(get_session)
):
    source = string.digits*4
    code = "".join(random.sample(source,4))

    message = MessageSchema(
        subject="【银月】注册验证码",
        recipients=[email],
        body=f"验证码为：{code}，有效期3分钟",
        subtype=MessageType.plain,
    )
    try:
        await mail.send_message(message)
    except SMTPResponseException as e:
        if e.code == -1 and b"\\x00\\x00\\x00" in str(e).encode():
            print("⚠️ 忽略 QQ 邮箱 SMTP 关闭阶段的非标准响应（邮件已成功发送）")

            email_code_repo = EmailCodeRepository(session = session)
            await email_code_repo.create(str(email),code = code)
        else:
            raise HTTPException(status_code=500,detail="邮件发送失败")
    return ResponseOut()
```
